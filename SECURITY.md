# Security Policy

## Supported Versions

Currently, only the active major version (v2.x) of the Digital Garden & Portfolio receives security updates. Legacy versions (v1.x) are no longer maintained.

| Version | Supported          |
| ------- | ------------------ |
| 2.5.x   | :white_check_mark: |
| 2.x.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

Security is taken very seriously. If you discover any security-related issues or vulnerabilities within this project (including API integrations, graph rendering, or data fetching scripts), please do not report them through public GitHub issues.

Instead, please report the vulnerability by emailing directly to:
**mailto:11220000064@ogrenci.ege.edu.tr**

### What to expect:
* You will receive an acknowledgment of your report within 48 hours.
* The issue will be investigated, and if a vulnerability is confirmed, a timeline for a fix will be provided.
* Once the issue is resolved, a security advisory will be published, and you will be credited for the discovery (if desired).

Thank you for helping keep this project safe and secure!
